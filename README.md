# Heart Disease Prediction System ❤️

## Overview
This project predicts whether a patient has heart disease using Machine Learning.

## Technologies Used
- Python
- Streamlit
- Flask
- Scikit-learn

## Features
- User-friendly interface
- Real-time prediction
- ML model integration

## Model
- Logistic Regression
- Accuracy: ~85%

## How to Run
1. Run backend: python app.py
2. Run frontend: streamlit run app.py
